Задание:
Реализовать компонент с применением технологий ядра D7, который выводит список зарегистрированных пользователей с постраничной навигацией.
Выборка должна кешироваться.
Постраничная навигация должна осуществляться Ajax-ом.
Сделать возможность выгрузки пользователей в csv и xml формат.

users.list - папка с файлами компонента
index.php - вызов компонента
